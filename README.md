# mengui

> 这是一个基于elementUI + vue实现的组件的二次封装
## 1、目录结构说明
```
├── dist                       # npm run build 生成的文件
├── src                        # 源代码
│   ├── assets                 # 样式和图片
│   ├── components             # 公用组件
├── App.vue                    # 入口html
└── package.json               # package.json
```
## 2、安装依赖与发布npm
``` bash
# 安装依赖
    npm install / yarn 
    建议使用yarn ，避免npm的丢包问题

# 发布组件
    1. npm run build / yarn build
    2. npm publish
```
